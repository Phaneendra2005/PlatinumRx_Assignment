# PlatinumRx Data Analyst Assignment

## Project Summary
End-to-end analytical solution across hospitality and healthcare domains using SQL, Excel, and Python.

## Tech Stack
- MySQL (Window Functions, CTEs)
- Excel (VLOOKUP, INDEX-MATCH, Pivot Tables)
- Python (Core logic, scripting)

## Approach

### SQL
- Built normalized schemas and sample datasets
- Implemented ranking logic using window functions
- Performed revenue, billing, and profitability analysis

### Spreadsheet
- Lookup operations using VLOOKUP and INDEX-MATCH
- Conditional metrics using COUNTIFS
- Pivot tables for aggregation

### Python
- Time conversion using integer arithmetic
- Duplicate removal preserving order without sets

## Key Insights
- Identified top revenue channels and customers
- Derived profit trends across clinics and cities
- Ranked items and bills to surface operational patterns

## How to Run

### SQL
Run files in MySQL Workbench or CLI:
- sql/hotel_system.sql
- sql/clinic_system.sql

### Python

Refer to excel/ directory for formulas and pivot steps
